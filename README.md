# GPO-Wallpaper

Created the GPO in Group Policy Management .<br>
Found an image in Pexel that had a JPG file type.<br>
Created a folder called "Wallpaper" that I moved the JPG file to.<br>
Shared the folder and copied the Network path.<br>
Edited the GPO and found the desktop policy under User Configuration\Policies\Administrative Temp\Desktop\Desktop.<br>
Pasted my networkpath to Wallpaper Name and pasted the file name.<br>
With cmd terminal I forced a group policy update.<br>
Logged off and signed back in to see the desktop wallpaper change.<br>
